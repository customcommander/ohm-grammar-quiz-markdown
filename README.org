* ohm-grammar-quiz-markdown

A grammar for converting a markdown-like document into a JSON-LD document:

#+begin_src markdown
What is the correct answer?

- [ ] 41
- [x] 42
- [ ] 42
#+end_src

becomes:

#+begin_src json
{
  "@context": "https://schema.org/",
  "@type": "Quiz",
  "hasPart": [
    {
      "@type": "Question",
      "text": "What is the correct answer?",
      "acceptedAnswer": {
        "position": 1,
        "@type": "Answer",
        "text": "42"
      },
      "suggestedAnswer": [
        {
          "position": 0,
          "@type": "Answer",
          "text": "41"
        },
        {
          "position": 2,
          "@type": "Answer",
          "text": "43"
        }
      ]
    }
  ]
}
#+end_src
